Desafio Bridge - Desenvolvimento Full Stack

A aplicação permite que o usúario digite um numero de sua escolha para descobrir se o número é primo e ver todos os seus divisores.

Tecnologias utilizadas:
O lado servidor, é desenvolvido em Node e Javascript utlizando o frameworks express.
Express é um framework para Node.js que contém um conjunto de recursos para desenvolvimento de aplicações web.

O frontend é desenvolvido em react e javascript e utiliza o axios para fazer a api.
O React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.
Axios é um cliente HTTP, sendo que a biblioteca é uma api que interage com a interface http do node.

Instruções
O usuario deve instalar o node na sua máquina, disponível em https://nodejs.org/en/
Após clonar o repositório, entre no diretório frontend e digite

'npm install'
em seguida digite
'npm start'

assim os pacotes Javascript utilizados na aplicação serão instalados

Depois entre no diretório backend e digite

'npm install'
em seguida digite
'node src/server.js'

para instalar os pacotes JavaScript utilizados pelo backend da aplicação.

E então a aplicação estará disponivel em
http://localhost:3000
